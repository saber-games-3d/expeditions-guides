# ReverseGear

Reverse Gear.

Attributes:

-   `AngVel ="2.0"`  
    The maximum angular speed of the wheel when this gear is active. Default value: `0`. Range of values: `[0.1; 32]`.


-   `FuelModifier ="0.9"`  
    The fuel consumption multiplier when this gear is active. Default value: `1`. Range of values: `[0; 10]`.
