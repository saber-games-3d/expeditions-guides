# UiDesc

Attributes:

-   `UiDesc="UI_WINCH_UPGRADE_SCOUT_DEFAULT_DESC"`  
    Description of the winch in the UI. Can be [localized][localized]. 

-   `UiName="UI_WINCH_UPGRADE_SCOUT_DEFAULT_NAME"`  
    Name of the winch in the UI. Can be [localized][localized].
