# SocketPoint

The `<SocketPoint>` tag describes the attachment point of the shaft. See [`<Shaft>`][shaft] for details.

Attributes:

-   `Name="Shaft1a"`  
    Name, used for the description of the shaft in the class of the truck.

-   `Pos="(1.595; 1.172; -0.004)"`  
    Position.
