# Zone Types in SnowRunner

*This topic is valid for SnowRunner only.*  

The list of all [**prop**](./../zone_types_overview.md) types available for zones in *SnowRunner* is below. 
It lists these props in the alphabetical order.

-   **ZonePropertyAutoRefuel** – see [AutoRefuel zones](./auto_refuel_zones.md).

-   **ZonePropertyAutoRepairAndRestore** – see [AutoRepairAndRestore zones](./auto_repair_and_restore_zones.md).

-   **ZonePropertyCargoLoading** – see [CargoLoading and ManualLoading zones](./cargoloading_and_manualloading_zones/cargo_loading_and_manual_loading_zones.md).

-   ***ZonePropertyCargoLoadingAllCargo*** – *Debug property. Similar to regular **ZonePropertyCargoLoading**, but allows the player to load all types of Cargo without limitations of their amount.*

-   ***ZonePropertyCargoLoadingAlwaysVisible*** – *Debug property. Similar to regular **ZonePropertyCargoLoading**, but makes the zone always visible and available to the player.*

-   **ZonePropertyEnergy** – see [StorehouseCraft and Energy zones](./storehouse_craft_and_energy_zones.md).

-   **ZonePropertyFarmingArea** – see [FarmingArea zones](./farming_area_zones.md).

-   **ZonePropertyFuelStation** – see [FuelStation zones](./fuel_station_zones.md).

-   **ZonePropertyGarageEntrance** – see [GarageEntrance and GarageExit zones](./garageentrance_and_garageexit_zones/garage_entrance_and_garage_exit_zones.md).

-   **ZonePropertyGarageExit** – see [GarageEntrance and GarageExit zones](./garageentrance_and_garageexit_zones/garage_entrance_and_garage_exit_zones.md).

-   **ZonePropertyGateway** – see [Regions with Multiple Maps][regions_with_multiple_maps] (for gateways between maps of the Region) and [Gateway zones][gateway_zones] (for gateways on the same map). 

-   **ZonePropertyLivingArea** – see [LivingArea zones](./living_area_zones.md).

-   **ZonePropertyManualLoading** – see [CargoLoading and ManualLoading zones](./cargoloading_and_manualloading_zones/cargo_loading_and_manual_loading_zones.md).

-   **ZonePropertyRecovery** – see [Recovery zones](./recovery_zones.md).

-   **ZonePropertyResupplyRepairParts** – see [ResupplyRepairParts zones](./resupply_repair_parts_zones.md).

-   **ZonePropertyStorehouseCraft** – see [StorehouseCraft and Energy zones](./storehouse_craft_and_energy_zones.md).

-   **ZonePropertyTrailerAttach** – see [TrailerAttach zones](./trailer_attach_zones.md).

-   **ZonePropertyUpgradesGiver** – see [UpgradesGiver zones](./upgrades_giver_zones.md).

-   **ZonePropertyWatchpoint** – see [Watchpoint zones](./watchpoint_zones.md).

-   **ZonePropertyWaterStation** – see [WaterStation zones](./water_station_zones.md).
